# MPred-IA - Frontend

## 📌 Descrição

Interface web desenvolvida em HTML, CSS e JavaScript puro para interação com a API MPred-IA.

Permite cadastrar equipamentos, registrar anomalias e visualizar os dados em tempo real.

---

## 🚀 Tecnologias utilizadas

* HTML
* CSS
* JavaScript (Vanilla JS)

---

## ⚙️ Funcionalidades

* Cadastro de equipamentos
* Cadastro de anomalias
* Listagem de equipamentos em formato de cards
* Exibição de anomalias
* Remoção de equipamentos
* Integração com API backend

```bash id="8hz2rm"
http://127.0.0.1:5000
```

Certifique-se de que o backend esteja em execução.

---

## 🧭 Como usar

1. Cadastre um equipamento
2. Informe o ID do equipamento para cadastrar uma anomalia
3. Visualize os equipamentos em cards
4. Veja as anomalias associadas
5. Remova equipamentos utilizando o botão "Deletar"
